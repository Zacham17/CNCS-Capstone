# Final Files Directory

This directory contains the completed files developed for this project.

The contents of each directory are outlined below:

<ins>**Demo Videos**:</ins>
This directory contains a downloadable video that demonstrates how the project works and shows how SIEM integration improves ticket resolution.


<ins>**Elastic YML Configs:**</ins>
This directory contains the configuration files for the components of the Elastic Stack

<ins>**ElasticData:**</ins>
This directory includes the scripts to query and parse through the elasticsearch data.

<ins>**ticketSite:**</ins>
This directory contains the Django Web application files and other files to build the ticketing system website
